# Simple Python Appplication

## Run with Docker

`docker build -t aqiapp .`  

`docker run -p 5000:8080 aqiapp`  

## Clean up

`docker container ls -a`  

`docker container rm [Container ID]`  

`docker rmi aqiapp:latest`
